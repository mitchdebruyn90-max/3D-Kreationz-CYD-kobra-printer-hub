# Arduino IDE Setup Guide for CYD UI\n\n## Prerequisites\n\n1. **Arduino IDE** (Version 1.8.19 or 2.0+)\n   - Download: https://www.arduino.cc/en/software\n\n2. **ESP32 Board Support**\n   - File → Preferences\n   - Add to \"Additional Boards Manager URLs\":\n   ```\n   https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json\n   ```\n   - Tools → Board Manager → Search \"ESP32\" → Install by Espressif\n\n3. **Required Libraries**\n   - Sketch → Include Library → Manage Libraries\n   - Search and install:\n     - `TFT_eSPI` (Bodmer)\n     - `ArduinoJson` (Bblanchon)\n     - `SdFat` (Bill Greiman)\n     - `AsyncTCP` (Me-no-dev)\n     - `ESPAsyncWebServer` (Me-no-dev)\n\n## Step 1: Configure Board Settings\n\n1. **Connect CYD via USB-C**\n\n2. **Select Board:**\n   - Tools → Board → ESP32 Arduino → \"ESP32 Dev Module\"\n\n3. **Configure Settings:**\n   ```\n   Board: ESP32 Dev Module\n   Upload Speed: 921600\n   CPU Frequency: 240 MHz\n   Flash Frequency: 80 MHz\n   Flash Mode: QIO\n   Flash Size: 4MB (32Mb)\n   Partition Scheme: Default 4MB with spiffs\n   Core Debug Level: None\n   PSRAM: Disabled\n   ```\n\n4. **Select Port:**\n   - Tools → Port → COM# (or /dev/ttyUSB0)\n   - If not visible, install CH340 drivers\n\n## Step 2: Configure TFT_eSPI Library\n\n1. **Locate TFT_eSPI Configuration:**\n   - Windows: Support\n\nCreate file `touch_input.h`:\n\n```cpp\n#ifndef TOUCH_INPUT_H\n#define TOUCH_INPUT_H\n\n#include <SPI.h>\n\n// XPT2046 Touch Controller Pins\nconst int TOUCH_CS = 33;\nconst int TOUCH_MOSI = 13;\nconst int TOUCH_MISO = 12;\nconst int TOUCH_SCLK = 14;\n\n// XPT2046 Commands\n#define XPT2046_X 0xD0\n#define XPT2046_Y 0x90\n\nstruct TouchPoint {\n    int x;\n    int y;\n    bool pressed;\n};\n\nclass TouchController {\npublic:\n    void begin() {\n        SPI.begin(TOUCH_SCLK, TOUCH_MISO, TOUCH_MOSI);\n        pinMode(TOUCH_CS, OUTPUT);\n        digitalWrite(TOUCH_CS, HIGH);\n    }\n    \n    TouchPoint readTouch() {\n        TouchPoint touch = {0, 0, false};\n        \n        digitalWrite(TOUCH_CS, LOW);\n        \n        // Read X\n        SPI.transfer(XPT2046_X);\n        uint16_t x = SPI.transfer16(0) >> 3;\n        \n        // Read Y\n        SPI.transfer(XPT2046_Y);\n        uint16_t y = SPI.transfer16(0) >> 3;\n        \n        // Check Z (pressure)\n        SPI.transfer(0xB0);\n        uint16_t z = SPI.transfer16(0);\n        \n        digitalWrite(TOUCH_CS, HIGH);\n        \n        if (z > 100) {  // Threshold for valid touch\n            touch.x = map(x, 0, 4095, 0, 320);\n            touch.y = map(y, 0, 4095, 0, 240);\n            touch.pressed = true;\n        }\n        \n        return touch;\n    }\n};\n\n#endif\n```\n\nInclude in main sketch:\n```cpp\n#include \"touch_input.h\"\n\nTouchController touch;\n\nvoid setup() {\n    // ... existing code ...\n    touch.begin();\n}\n\nvoid loop() {\n    TouchPoint tp = touch.readTouch();\n    if (tp.pressed) {\n        Serial.print(\"Touch: \");\n        Serial.print(tp.x);\n        Serial.print(\", \");\n        Serial.println(tp.y);\n    }\n    delay(50);\n}\n```\n\n## Step 6: Add Color Themes\n\nCreate `colors.h`:\n\n```cpp\n#ifndef COLORS_H\n#define COLORS_H\n\n#include <TFT_eSPI.h>\n\n// Theme: Ocean\nnamespace Theme_Ocean {\n    const Check USB connection\n2. Verify board is ESP32 Dev Module\n3. Check TFT_eSPI User_Setup.h configuration\n4. Try pressing RESET button on CYD\n\n### Upload Fails\n1. Install CH340 drivers (Windows/Mac)\n2. Select correct COM port\n3. Try lower upload speed (115200)\n4. Hold BOOT button while uploading\n\n### Touch Not Working\n1. Check XPT2046 chip on back of board\n2. Verify touch CS pin (GPIO 33)\n3. Test with multimeter for shorts\n\n### Slow Performance\n1. Increase SPI frequency in User_Setup.h\n2. Reduce refresh rate (increase delay)\n3. Check for Serial.print() overhead\n4. Profile with micros()\n\n## Next Steps\n\n1. **Implement WiFi Connectivity:**\n   - Add WiFi library\n   - Connect to printer\n   - Display network status\n\n2. **Add SD Card Support:**\n   - Initialize SD card\n   - Read/write files\n   - Display file browser\n\n3. **Create App Screens:**\n   - Print monitor\n   - File manager\n   - 3D modeler preview\n\n4. **Add Real-time Updates:**\n   - Temperature graphs\n   - Progress bars\n   - Status indicators\n\n## Resources\n\n- TFT_eSPI GitHub: https://github.com/Bodmer/TFT_eSPI\n- ESP32 Board Package: https://github.com/espressif/arduino-esp32\n- CYD Pinout: https://github.com/Sunton-Tech/ESP32-2432S028R\n- XPT2046 Library: https://github.com/PaulStoffregen/XPT2046_Touchscreen\n"
